h# Mini 2D RPG Exploration Demo

A static browser prototype inspired by classic top-down RPG Maker exploration.

## How to Play

Open `index.html` in a modern browser.

Controls:

- `W`, `A`, `S`, `D`: move in eight directions
- `F`: talk to a nearby NPC or advance dialogue

## Features

- One large peaceful fantasy village map
- Smooth camera follow with world-boundary clamping
- Rectangular collision for houses, trees, rocks, water, NPCs, and map edges
- Six fixed NPCs with 2-3 dialogue pages each
- RPG-style bottom dialogue window
- Static HTML, CSS, and JavaScript only

No backend, build tools, Node.js, account system, save system, combat, inventory, or quest mechanics are included.
